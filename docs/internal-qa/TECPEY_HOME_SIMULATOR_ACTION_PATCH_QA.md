# TecPey Home Simulator Action Patch QA

## هدف پچ
کارت «شبیه‌ساز تصمیم» در صفحه Home نباید کاربر را به آکادمی عمومی ببرد یا بدون عملکرد بماند. باید مستقیماً وارد تمرین عملی شبیه‌ساز شود.

## تغییرات انجام‌شده
- کارت «شبیه‌ساز تصمیم» در Home فارسی به `/academy/simulator` لینک شد.
- کارت `Decision Simulator` در Home انگلیسی به `/en/academy/simulator` لینک شد.
- برای کارت‌ها CTA کوچک اضافه شد؛ برای شبیه‌ساز متن CTA به «شروع شبیه‌ساز / Start simulator» تغییر کرد.
- لینک کارت‌های دیگر نیز اختصاصی‌تر شدند:
  - مربی هوشمند → `/academy/ai-guide`
  - آکادمی مرحله‌ای → `/academy`
  - امنیت قبل از معامله → `/academy/term-2`
- نسخه انگلیسی هم معادل‌سازی شد.

## تست پیشنهادی
```bash
npm install
npm run build
npm start
```
سپس در Home روی کارت «شبیه‌ساز تصمیم» کلیک شود و باید به صفحه تمرین تصمیم بازار برود، نه آکادمی عمومی.
