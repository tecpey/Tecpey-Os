# TecPey Academy Product Polish / Red Team Patch

## هدف
حذف حس پروتوتایپ، پاک‌سازی متن‌های رو به سازنده، جلوگیری از فاش شدن رودمپ/چت داخلی و تبدیل صفحات آکادمی، داشبورد دانشجو و شبیه‌ساز به تجربه محصولی قابل ارائه.

## اصلاحات اصلی
- متن‌های داخلی مثل MVP، Roadmap اجرایی، DATABASE_URL، نسخه عملیاتی، پاسخ به ترس Down شدن و اصطلاحات سازنده از UI عمومی حذف یا محصولی شدند.
- Student Dashboard از حالت کارتکس فنی/داخلی به «پرونده یادگیری دانشجو» تبدیل شد.
- وضعیت‌های Sync دیگر جزئیات فنی مثل PostgreSQL/DATABASE_URL را به کاربر نشان نمی‌دهند.
- شبیه‌ساز از «نقشه آینده» به «تمرین تصمیم‌گیری بازار بدون پول واقعی» تبدیل شد.
- Case Study Lab از متن توجیهی برای سازنده به تمرین عملی کاربر تبدیل شد.
- AI Mentor demo دیگر API فعال/غیرفعال و Guardrails توسعه‌ای را فاش نمی‌کند.
- بخش WorldClassUpgrade از Operating System/Scale Board به مسیر رشد دانشجو و زیرساخت پایدار تک‌پی تبدیل شد.

## ریسک باقی‌مانده
- برخی فایل‌های انگلیسی هنوز کلمه roadmap را در معنای طبیعی مسیر یادگیری/پروژه استفاده می‌کنند، نه به‌عنوان رودمپ سازنده.
- Build کامل باید در محیط دارای node_modules با `npm ci && npm run build` انجام شود.
