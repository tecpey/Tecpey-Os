# TecPey Engineering Records

This directory contains durable engineering governance, planning, audit, workspace, and readiness records.

## Structure

- `governance/`: permanent engineering rules and operating standards.
- `phase39/`: Phase 39 and Phase 39.5 audit, planning, and task-board records.
- `wallet/`: wallet-specific readiness and risk reports.
- `workspace/`: repository hygiene and working-tree cleanup records.

## Rules

- Keep implementation code out of this directory.
- Keep phase plans, task boards, audit reports, and readiness reports under the closest matching subfolder.
- Do not store secrets, screenshots with sensitive data, local archives, or personal artifacts here.
- When moving engineering records, update cross-references in the affected documents.
